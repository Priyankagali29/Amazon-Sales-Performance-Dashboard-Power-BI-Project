This project focuses on transforming raw business data into meaningful, decision-driven insights using interactive visualization techniques.

Key Project Components:
 • Business Requirement Analysis & KPI Identification
 • Data Cleaning & Pre-Processing
 • Data Modeling & Relationship Building
 • Customer Behavior Analysis (Ratings, Reviews, Discount Impact)
 • Trend Analysis (Year-over-Year Performance)
 • Interactive Dashboard Development
 • Business Insight Generation & Storytelling

Dashboard Interactivity Features:
 • Global filters for Date, Product Category, Region, and Payment Method
 • Cross-filtering across visuals
 • Drill-down for time-based trend analysis
 • Drill-through navigation between overview and deep analysis pages

Design Principles Followed:
 • Clean corporate theme
 • Consistent color palette
 • High contrast for readability
 • Logical and structured layout
 • Minimal visual clutter for better decision-making

Tools & Skills Used:
 Power BI | Data Modeling | Data Visualization | Business Intelligence | 
 Data Analysis
